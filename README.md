SQL Assistant Bot 🤖

An AI-powered SQL Assistant built using Streamlit and Groq.

Features

- Convert natural language into SQL queries
- Supports MySQL, PostgreSQL, SQLite, and SQL Server
- Query explanations
- Chat history
- Download generated SQL
- Clean Streamlit UI

Tech Stack

- Python
- Streamlit
- Groq LLM
- LangChain
- GitHub

Run Locally

pip install -r requirements.txt
streamlit run app.py
